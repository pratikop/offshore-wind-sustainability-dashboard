# Data Sources

This project uses publicly available sustainability data from RWE's official 2025 reporting.

## Main sources

- RWE Annual Report 2025, Group Sustainability Statement
- RWE Sustainability Key Figures Report 2025
- RWE Group Sustainability Reporting – Data 2025

## Data scope

- The dataset uses RWE Group-level sustainability data.
- It is not project-level offshore wind farm data.
- No confidential or internal RWE data is used.
- The dashboard is created for educational and portfolio purposes.

## Notes

Some biodiversity entries are target-based indicators because biodiversity impact is reported as strategic goals rather than simple annual numeric KPIs.
